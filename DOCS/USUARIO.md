# 📕 Guía de Usuario: ENSING CRM v2.0

Esta guía explica las nuevas funcionalidades y cómo sacarles el máximo provecho para tu flujo de ventas.

## 1. Gestión de Leads de Alto Impacto
Al abrir un lead, ahora verás un panel lateral optimizado:
- **Información Central**: El valor del negocio y la probabilidad (Score AI) aparecen en grande para que no tengas que buscarlos.
- **Etiquetas Inteligentes**: Puedes escribir etiquetas separadas por comas (ej: `VIP, Software, Urgente`) y el sistema las guardará automáticamente.
- **Navegación Fluida**: Puedes scrollear por el contenido mientras mantienes la cabecera visible.

## 2. Continuidad de Trabajo (Persistencia)
El CRM ahora "recuerda" por ti:
- Si estás en la página de **Empresas** y recargas la página, volverás a **Empresas**.
- Si estabas viendo un **Pipeline específico** (ej: Funnel de Renovaciones), el sistema lo mantendrá seleccionado para ti.

## 3. Automatización Paso a Paso
En el módulo de **Automatizaciones**:
- **CREAR REGLA**: Haz clic en el signo `+` de cualquier etapa para añadir una acción automática.
- **MODO PRUEBA**: Actívalo arriba a la derecha para probar tus flujos sin miedo a enviar correos reales.
- **INSPECTOR**: Haz clic en cualquier regla para ajustar las condiciones de disparo o el mensaje.

## 4. Búsqueda Global (Spotlight)
Presiona `Ctrl + K` (o `Cmd + K`) en cualquier momento para abrir el buscador global. Desde ahí puedes saltar rápidamente a cualquier módulo o cerrar sesión.
